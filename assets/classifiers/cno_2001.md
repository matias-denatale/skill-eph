**Fuente:** INDEC — Encuesta Permanente de Hogares
**Versión:** CNO-2001 (versión reducida basada en CNO-PROMAEO)
**Variable en microdatos EPH:** `PP04D_COD` (5 dígitos)

---

## Estructura del CNO-2001

El CNO-2001 ordena en forma sistemática las ocupaciones del total de la población activa mediante **5 dígitos**, cada uno representando una dimensión analítica:

```
XX . X . X . X
│    │   │   └─ 5° dígito: Calificación Ocupacional
│    │   └───── 4° dígito: Tecnología Ocupacional
│    └───────── 3° dígito: Jerarquía Ocupacional
└────────────── 1° y 2° dígito: Carácter Ocupacional
```

### 3° Dígito — Jerarquía Ocupacional

Distingue las formas jerárquico-organizativas del proceso de trabajo.

| Valor | Jerarquía |
|-------|-----------|
| 0 | Dirección |
| 1 | Cuenta propia |
| 2 | Jefes |
| 3 | Trabajadores asalariados |

---

### 4° Dígito — Tecnología Ocupacional

Identifica el uso de formas tecnológicas en el proceso de trabajo.

| Valor | Tecnología |
|-------|-----------|
| 1 | Sin operación de máquina |
| 2 | Operación de maquinaria y equipos electromecánicos |
| 3 | Operación de sistemas y/o equipos informatizados |

---

### 5° Dígito — Calificación Ocupacional

Mide la complejidad del proceso de trabajo y sus requerimientos de conocimientos y habilidades.

| Valor | Calificación |
|-------|-------------|
| 1 | Profesionales |
| 2 | Técnicos |
| 3 | Operativo |
| 4 | No calificado |

## Lectura de un código CNO completo

**Ejemplo:** `10.3.1.4`
- `10` → Carácter: gestión administrativa, planificación y control de gestión
- `3` → Jerarquía: trabajador asalariado
- `1` → Tecnología: sin operación de máquina
- `4` → Calificación: no calificado
→ **Trabajador asalariado, sin maquinaria, no calificado, en gestión administrativa**

**Ejemplo:** `81.1.3.1`
- `81` → Carácter: producción de software
- `1` → Jerarquía: cuenta propia
- `3` → Tecnología: operación de sistemas y/o equipos informatizados
- `1` → Calificación: profesional
→ **Desarrollador de software independiente, altamente calificado**

**Ejemplo:** `55.3.1.4`
- `55` → Carácter: servicios domésticos
- `3` → Jerarquía: trabajador asalariado
- `1` → Tecnología: sin operación de máquina
- `4` → Calificación: no calificado
→ **Empleada doméstica asalariada, sin calificación**
